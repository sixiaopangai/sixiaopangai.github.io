## 🛠️ 技术栈

这个博客基于以下技术构建，体现了我对 Web 开发及 3D 图形的兴趣：

| 技术 | 说明 |
|------|------|
| **框架** | [Hexo](https://hexo.io/) - 静态网站生成器 |
| **主题** | [Fluid](https://github.com/fluid-dev/hexo-theme-fluid) - 优雅的 Material Design 主题 |
| **交互** | [Three.js](https://threejs.org/) - 全站动态粒子背景 |
| **部署** | GitHub Pages + [Cloudflare](https://www.cloudflare.com/) - CDN 加速与域名管理 |
| **环境** | 开发于 Acer Predator PHN16-71 |

---

## ✨ 核心特性

- 🎨 **动态视觉**：注入了自定义的 Three.js 粒子系统，支持随鼠标滑动的交互效果
- 📱 **响应式设计**：PC 端显示动态粒子，移动端显示静态图片以优化性能
- 📝 **全栈记录**：包含 Spring Boot、Vue.js、Three.js 以及 VPS 运维的相关笔记（可能吧）
- 🔢 **数学支持**：支持 LaTeX 公式渲染，用于记录工程经济学等学术笔记（可能吧）
